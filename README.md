
# 🚀 Plataforma de Cursos Tech — Landing Page com Checkout (Projeto Fictício)

Este projeto consiste no desenvolvimento de uma **landing page de vendas para cursos na área de tecnologia**, com simulação de fluxo de compra e foco em **experiência do usuário, interatividade e responsividade**.

A aplicação foi construída **sem uso de frameworks**, com o objetivo de consolidar fundamentos de desenvolvimento front-end e lógica de interface.

---

## 🎯 Objetivo do Projeto

* Praticar desenvolvimento web com **HTML, CSS e JavaScript puro**
* Aplicar conceitos de **responsividade (Mobile First)**
* Simular um fluxo real de checkout
* Trabalhar organização de código e estrutura semântica
* Desenvolver habilidades colaborativas em equipe

---

## 🛠️ Stack Utilizada

* **HTML5** → Estrutura semântica e organização do conteúdo
* **CSS3** → Layout responsivo, flexbox e estilização
* **JavaScript (Vanilla JS)** → Interatividade, manipulação do DOM e validações de formulário

---

## 💡 Funcionalidades Implementadas

* Layout totalmente responsivo (desktop, tablet e mobile)
* Sessão de apresentação do curso com proposta de valor
* Seção de benefícios e informações estratégicas
* Botões com interações dinâmicas
* Simulação de checkout
* Validação básica de campos no formulário de pagamento
* Organização modular de scripts

---

## 📈 Próximos Passos (Roadmap Técnico)

O projeto será evoluído para uma estrutura mais próxima de um ambiente real de produção:

* Integração com API de pagamentos
* Implementação de backend (Node.js ou outra tecnologia)
* Criação de banco de dados para usuários e pedidos
* Autenticação de usuário
* Persistência de dados
* Deploy completo com ambiente estruturado

---

## 🧠 Competências Demonstradas

* Manipulação do DOM
* Lógica de validação de formulários
* Responsividade com CSS
* Organização de projeto front-end
* Trabalho em equipe e versionamento
